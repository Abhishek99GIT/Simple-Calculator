CALCULATOR

This is Abhishek Kolloju. I created a basic calculator that could be shown in a browser using HTML, CSS-cascading style sheet, and basic java script. I have defined a calculator container and positioned it in the middle. To show the numbers, I installed a display with input type. Next, in order to affect the display value, I added buttons, each of which had a onclick value. In order to make the calculator seem nice, I added the CSS required. In order to obtain the evaluation of the string in the display value, I ultimately used the eval() method on toString() of the display value at the onclick button of Equal to.
